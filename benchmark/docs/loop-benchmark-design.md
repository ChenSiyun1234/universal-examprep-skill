# 备考全流程 Benchmark（v4 设计冻结稿）

> 取代「单题问答」式旧测法的设计文档。旧三臂矩阵**保留**用作回归与正确率/弃答的度量，
> 但 README 的主叙事换成本文的**全流程六项**——因为技能的主要价值在「整个备考循环」，
> 而单题问答恰好是裸智能体也能凑合的那一小块（v3 实测裸文件臂正确率能追平技能臂）。

## 设计原则

1. **测真实痛点，不测公平话术**。六项全部对应学生真实会遇到的坑；裸智能体在其中四项
   是**结构性缺失**（不是发挥不好，是根本没有这个机器）——差距因此天然拉开，无需操纵。
2. **能确定性判分的绝不用 LLM 裁判**。六项中四项由脚本直接量出（解析回答/检查磁盘/
   校验锚点/数 PDF 页），免判分偏差、免裁判成本；只有正确率与弃答沿用既有 LLM 判分
   （已有 κ=0.833/0.875 人工校准背书）。
3. **一句话能讲明白**。每项一个白话名字 + 一个百分数，README 一张表 + 一张图放完。

## 六项定义（无 skill 臂 vs 有 skill 臂）

| # | 项目（README 用名） | 痛点 | 度量方法 | 判分 |
|---|---|---|---|---|
| 1 | **句句有出处**（溯源可核验率） | AI 讲得自信，无从核对 | 教学回合的回答里，带来源标注且**锚点真实可解析**（文件存在+锚在目标里）的结论占比 | 确定性（解析+核锚） |
| 2 | **不会瞎编**（越界弃答率） | 材料没有的也硬答 | 沿用越界探针（每课 10 道材料无答案的题），如实说「材料里没有」的占比 | LLM 判 + 弃答词表 |
| 3 | **材料里有就答对**（grounding 正确率 + 检索命中） | 材料里有但模型不知道 | 沿用材料专属难题集（PSYC 54 + 6.006 65）；新增：轨迹记录下的**章路由命中率 recall@1** | LLM 判（既有校准）+ 确定性（轨迹） |
| 4 | **关掉聊天不丢东西**（跨会话存续率） | 聊天记录一关全蒸发 | 会话 A 教学 3 题+测验 2 题（1 题故意答错）→ **全新会话 B**（零上下文）问「上次错了哪题」「把上次讲的 X 再给我」；B 能准确复原的会话 A 事实占比 | 确定性为主（错题 id/结论内容匹配），边缘 LLM 复核 |
| 5 | **复习完手里有东西**（交付物完备性） | 复习一晚啥也没留下 | 循环结束时清点：笔记本条目≥教学数且目录锚全通、错题本含答错题的完整条目、小抄每条要点溯源通过 validate、PDF 恰为指定页数 | 确定性（validate_workspace + pdf_page_count） |
| 6 | **省上下文省钱**（成本效率） | 每题都翻整堆材料 | 沿用每题 $ 成本 + 每会话 token；skill 臂应显著低于裸 dump | 确定性（API 计费字段） |

**预期格局（诚实声明）**：第 1/4/5 项裸智能体接近 0%（无锚点体系/无跨会话存储/无产物契约）——
这不是「打压对照组」，而是这些能力**正是技能存在的理由**；第 2/3/6 项延续 v3 的真实差距。
README 呈现时保留这句说明，防「操纵基准」质疑。

## 协议（loop_bench.py）

- **臂**：`bare`（同一材料文件夹 + 通用助手提示，无技能文件）vs `skill`（v4 完整工作区 + 技能包）。
  两臂同模型、同题目、同会话脚本。
- **会话模拟**：多轮 = 逐轮 `claude -p`，把此前轮次的（用户话语+助手回答）作为转写前缀拼进
  prompt（确定性、可断点续跑）；「全新会话」= 清空前缀、仅保留磁盘工作区。
- **会话脚本**（每课程一遍）：
  S1 教学会话：教 3 道金标题（七步/自由格式各按臂）→ 测 2 道（模拟学生第 2 题答错）→ 结束。
  S2 存续会话（新上下文）：「上次我错了哪几道？」「把上次讲过的 <q1> 结论再给我看一眼」。
  S3 小抄会话：「给我 2 页的考前小抄 PDF」。
- **课程**：沿用本地已建的 psyc110_full 与 mit6006_full 工作区（bare 臂用其原始材料目录）。
- **模型**：主跑 sonnet（成本可控）；矩阵回归照旧三模型。
- **判分器**：`loop_score.py` 纯标准库——溯源解析（源块正则+锚核验复用 validate 的 `_md_anchors`）、
  存续匹配（错题 id 集合、结论关键词覆盖）、交付物清点（validate_workspace + `pdf_page_count`）。
- **--mock**：假转写夹具端到端跑通全管线（CI 常驻，零成本）。

## 回归矩阵（并行进行）

既有 `run_matrix.py --real` 三臂×三模型跑 v4 工作区 + `EXAMPREP_TRACE=1` 出章路由 recall；
对照 v3 基线（REPORT 所载）。验收：材料专属正确率与越界弃答不低于 v3；recall@1 为新增亮点。
