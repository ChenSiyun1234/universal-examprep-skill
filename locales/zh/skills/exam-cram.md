# exam-cram — zh 学生侧文案包

> 本文件是学生可见文案的 zh 语言包；行为逻辑在 [skills/exam-cram/SKILL.md](../../../skills/exam-cram/SKILL.md)（控制层，单一事实源）。

## Student-facing Output

`中文` 模式（及 `双语` 的 `zh` 侧）学生侧使用规范中文词表（当前阶段 / 这题考什么 / 标准答题步骤 / 易错点 / 3分钟速记 / 现在轮到你 / 已记录到错题本 / 必背 / 例题 / 例题解答 / 要点解释 / 错题重做 / 疑难复述 / 已初始化备考空间）；`English` 模式使用英文规范词表（派发规则见控制层 `Output Contract` 与 [`docs/language-policy.md`](../../../docs/language-policy.md)）。`中文` 模式下来源标注逐字呈现为：

- 🟢 **来自资料**：直接源自学生上传内容，可信度高。
- 🟡 **AI 补充**：资料未覆盖、AI 用自身知识补的，标注「🟡 AI补充，可能与你老师讲的不完全一致」（以老师为准）。
- ⚠️ **AI 生成答案**：老师只勾题没给答案时 AI 代答的，每个都标「⚠️ AI生成答案，非老师/教材提供」。

学生可见输出默认英文（学生用中文开场则简体中文）；已持久化的 `language` 按控制层 `Output Contract` 派发规则切换，各模式单语言纯净。

双语组合规则（`language=双语`）：绝不使用第三套模板——逐块组合：先出 `zh` 块（纯中文、中文规范形），紧跟一行 `> EN:` 英文镜像（纯英文、英文规范词表）；两侧各自单语言纯净、每个锚点每侧只出现一次。进度面板、回执、来源块也逐行同法镜像。`≤1天` 档 `> EN:` 镜像可压缩为关键句（时间优先于完整）。
